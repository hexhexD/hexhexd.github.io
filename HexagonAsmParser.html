Classes and functions

HexagonAsmParser

  • It inherits from MCTargetAsmParser which has MatchAndEmitInstruction function to be called for instruction parsing and emitting.
    1. MatchAndEmitInstruction first checks if the current line is already in a packet

HexagonOperand

  • Must be defined before instruction matcher can be created
  • Represents a parsed instruction storing types and contents of its operands.
  • Corresponds to OperandVector parameter in MatchAndEmitInstruction function
