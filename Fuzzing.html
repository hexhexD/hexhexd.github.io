Sandsifter

  • Instruction length varies. Hexagon has sub-insturctions, duplex, packets.
  • Determine length of an arbitrary insturction using page fault.
  • Cover all kinds of instructions includes kernel, hypervisor, or system management code. Using esceptions.
  • Using #UD to filter out undefined opcode.
  • Protect processor state while fuzzing. We dont want it to crash. Hook exeception handler to restore fuzzer to a good state. Make all register 0.
  • Using a dissassembler to find anomalies

UISFuzz

  • Using similar strategies as Sandsifter
